# 🤖 Braço Robótico & Sistema de Sensores para Monitoramento Ambiental

## 📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de um **braço robótico com garra funcional** aliado a um **sistema de sensores ambientais**, com o objetivo de simular operações automatizadas e monitoramento contínuo de condições do ambiente.

O braço robótico é capaz de executar movimentos controlados e manipular objetos, enquanto os sensores coletam dados como temperatura e força, exibindo-os em um display LCD.

## 🔧 Especificações Técnicas — Braço

* Arduino Uno
* 2 Servomotores (pinos 9 e 10)
* LED de status (pino 7)
* Alimentação: 5V

---

## 🧩 Modelagem 3D

* Software utilizado: **OpenSCAD**
* Estrutura parametrizada, permitindo ajustes dimensionais rápidos
* Design voltado para fácil adaptação e prototipagem

## 🤖 Função da Garra

A garra robótica foi projetada para:

* Conectar uma **mangueira de material**
* Auxiliar no **preenchimento de estruturas**
* Simular processos automatizados industriais
